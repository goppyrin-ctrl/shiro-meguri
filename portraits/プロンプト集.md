# 武将イラスト生成プロンプト集（全249人）

## 使い方

1. ChatGPT・Gemini・Copilot などの画像生成AIに、下のプロンプトを **1ブロックずつコピペ** して生成する
2. 気に入った画像を **「保存名」のファイル名**（例: `伊達政宗.png`）で保存する（png / jpg / webp どれでも可。正方形でもOK＝ゲーム側で自動トリミング）
3. `デスクトップ → shiro-meguri → portraits` フォルダに入れる
4. ゲームを開き直すと、その武将だけAIイラストに差し替わる（無い武将は今まで通りの似顔絵）
5. 画像が溜まったらClaude（私）に「画像入れたよ」と言えば、圧縮して公開版にも反映します

**コツ:** 同じチャットの中で続けて生成すると画風が揃いやすい。まずは下の「レア大名」17人からがおすすめ（一番目立つ場所に出ます）。
